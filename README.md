# Portfolio Website — Muhammed Sahad S

A personal portfolio website built using HTML5 and CSS3.

---

## Project Overview

A fully responsive single-page portfolio website designed to showcase my skills, projects, and contact information. The site features an earthy warm design aesthetic with CSS animations, hover effects, smooth scrolling, and a custom layout built entirely with HTML and CSS.

**Sections included:**
- Home / Hero
- About Me
- Skills
- Projects
- Contact

**Features used:**
- CSS Grid layout
- Flexbox layout
- Hover effects and transitions
- CSS animations (float, pulse)
- Smooth scrolling
- Gradient backgrounds
- Card layouts
- Box shadows
- Responsive images
- Custom CSS variables
- Fluid typography using clamp()
- Pseudo-elements for decorative styling
- Backdrop filter on navigation
- Custom scrollbar styling
- Google Fonts (Playfair Display + DM Sans)

---

## Tech Stack

- HTML5
- CSS3 (External stylesheet)

## Setup Instructions

1. Clone the repository:
```
   git clone https://github.com/sahad-s/Portfolio-website-.git
```
2. Navigate into the project folder:
```
   cd portfolio-website
```
3. No dependencies or installations required.
